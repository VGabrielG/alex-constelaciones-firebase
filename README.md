# Alexandros Tefarikis - Portafolio Musical

Este sitio web es el portafolio profesional de Alexandros Tefarikis, músico, compositor y productor de eventos. Construido con Angular 20 y Firebase.

## 🚀 Inicio Rápido

Para iniciar el servidor de desarrollo local:
```bash
npm install
npm start
```
Luego navega a `http://localhost:4200/`.

## 🛠️ Tecnologías Principales
- **Framework**: [Angular 20+](https://angular.dev/)
- **Hosting**: [Firebase](https://firebase.google.com/)
- **Gestión de Estado**: Angular Signals
- **Estilos**: CSS nativo con diseño responsivo.

## 📁 Estructura del Proyecto
- `src/app/`: Lógica de la aplicación y componentes.
- `public/`: Activos estáticos (imágenes, favicons).
- `src/styles.css`: Estilos globales base.
